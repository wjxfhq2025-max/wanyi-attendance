## 🚀 下一步：在 Render.com 部署

### 第一步：创建 PostgreSQL 数据库

1. 打开浏览器，访问 **https://render.com**
2. 用 GitHub 账号登录（或注册）
3. 点击 **New** → **PostgreSQL**
4. 配置：
   - **Name**: `wanyi-attendance-db`
   - **Region**: Oregon (US West) 或 Singapore
   - **PostgreSQL Version**: 15
   - **Plan**: **Free**
5. 点击 **Create Database**
6. 创建后，复制 **Internal Database URL**（类似：`postgresql://...`）

### 第二步：创建 Web Service

1. 点击 **New** → **Web Service**
2. 点击 **Connect a repository**
3. 选择 `wjxfhq2025-max/wanyi-attendance` 仓库
4. 配置：

| 设置项 | 值 |
|--------|-----|
| **Name** | `wanyi-attendance` |
| **Region** | 与数据库相同 |
| **Branch** | `master` |
| **Root Directory** | （留空） |
| **Runtime** | `Node` |
| **Build Command** | `npm install` |
| **Start Command** | `node server.js` |
| **Plan** | **Free** |

### 第三步：添加环境变量

在 **Environment Variables** 部分，添加：

| Key | Value |
|-----|-------|
| `DATABASE_URL` | （粘贴刚才复制的 Internal Database URL） |
| `NODE_ENV` | `production` |

### 第四步：部署

1. 点击 **Deploy Web Service**
2. 等待几分钟，查看部署日志
3. 成功后会显示域名，如：`https://wanyi-attendance.onrender.com`
